# Features and Workflow

AIdea is built around a research-native workflow inside Zotero.

## Side Panel Chat

You can chat with AI directly in:

- the library side panel
- the PDF reader side panel

This keeps questions, notes, and source material in one place.

## Paper-Aware Context

When you select text from a PDF and add it to the context area, AIdea can answer questions with that passage in mind. This is useful for:

- explaining dense paragraphs
- checking claims against the paper text
- comparing methods or results sections
- translating a selected excerpt

## Quick Actions

AIdea includes shortcut actions for common paper tasks, such as:

- summarize
- explain
- translate
- extract key points
- identify limitations

These shortcuts can be customized to fit different reading styles.

## File and Image Input

AIdea supports multimodal workflows, including images such as figures, screenshots, and charts. This is helpful when discussing visual content from a paper.

## Note Export

Responses can be exported into Zotero notes. The output supports:

- Markdown structure
- code blocks
- tables
- LaTeX math rendering

## Persistent Chat History

Conversations are stored locally so users can revisit previous threads instead of starting from scratch every time.

## Local Memory System

AIdea can retain useful context across conversations while keeping storage local to Zotero. The memory system is designed to improve continuity without sending a separate memory database to external services.

## Best Workflow Tips

- Use quick actions for repetitive tasks
- Add selected PDF text before asking precise questions
- Export high-value answers into Zotero notes
- Keep different research libraries separate to benefit from scoped local memory
