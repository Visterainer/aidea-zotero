# Getting Started zh-CN

这一页提供从安装到第一次成功对话的最短路径。

## 1. 安装 AIdea

1. 从仓库的 Releases 页面下载最新版本。
2. 在 Zotero 中打开 `工具 -> 插件`。
3. 点击齿轮菜单，选择 `从文件安装插件...`。
4. 选中下载好的 `.xpi` 文件并安装。
5. 重启 Zotero。

## 2. 打开设置页

重启后，在 Zotero 中打开 AIdea 设置：

- `工具 -> 插件 -> AIdea -> 设置`
- 某些环境下也可能在 `编辑 -> 设置 -> AIdea`

## 3. 先配置一个服务商

在服务商卡片上，推荐按下面顺序操作：

`Install/Update Env -> OAuth Login -> Refresh Models`

补充说明：

- OpenAI 和 Gemini 需要先执行环境安装。
- Qwen 和 GitHub Copilot 不需要额外运行时安装。
- 登录完成后，记得点击 `Refresh Models` 读取账号可用模型。

## 4. 打开 AI 面板

AIdea 主要可以在两个位置使用：

- Zotero 文库右侧边栏
- PDF 阅读器右侧边栏

选择一个条目或打开 PDF 后，在右侧找到 AIdea 面板。

## 5. 发送第一条消息

可以先试试这些问题：

- “请用 5 个要点总结这篇论文。”
- “请用通俗语言解释这篇论文的核心方法。”
- “作者提到的主要局限性是什么？”

## 6. 添加带依据的上下文

在 PDF 阅读器中：

1. 选中一段文字。
2. 点击 `Add Text`。
3. 再发送与这段文字有关的问题。

这样模型会更容易基于论文原文给出回答。

## 7. 使用快捷操作

AIdea 内置了常见研究任务的快捷按钮，比如总结、翻译、提取重点等。你也可以按自己的工作流进行调整。

## 如果遇到问题

- 检查当前服务商是否已经登录
- 点击 `Refresh Models`
- 确认 Zotero 版本是 7 或以上
- 完成配置后重启一次 Zotero
- 参考 [[FAQ zh-CN]]
