# FAQ zh-CN

## 需要 API Key 吗？

通常不需要。AIdea 主要通过 OAuth 登录流程工作，用户可以直接使用已有账号。

## 支持哪些服务商？

支持 OpenAI (ChatGPT)、Google Gemini、Qwen 和 GitHub Copilot。

## 需要安装 Node.js 吗？

只有部分服务商流程需要，主要是 OpenAI 和 Gemini。插件可以自动安装所需环境。Qwen 和 GitHub Copilot 不需要这个额外步骤。

## 聊天记录存在哪里？

聊天记录保存在 Zotero 的本地数据库中。

## 认证令牌存在哪里？

认证数据保存在用户本机。

## AIdea 会收集用户数据吗？

项目说明中表示它不会收集用户数据，模型请求会直接发送给用户选择的服务商。

## 为什么登录后看不到模型？

先尝试点击 `Refresh Models`。如果还不行，可以移除认证后重新登录。同时也要确认你的服务商账号本身拥有相应模型的访问权限。

## 为什么回答没有很好地基于论文内容？

如果问题与论文某一段密切相关，最好先在 PDF 中选中那一段并使用 `Add Text`，再发起提问。

## 只能用于英文论文吗？

不是。AIdea 支持英文和中文界面，也适合用于多语言阅读和翻译场景。

## AIdea 是开源的吗？

是。这个项目在 GitHub 上公开发布，许可证是 `AGPL-3.0-or-later`。
